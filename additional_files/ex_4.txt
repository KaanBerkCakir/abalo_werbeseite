Hilfreiche Features in PHPStorms:

1. Strg + W
   Erleichert das Auswählen von Codefragmenten enorm. Dabei wird beim ersten Mal
   das Wort markiert, wo sich der Cursor gerade befindet. Danach der gesamte Ausdruck usw.

2. Einbindung von VSC
   Man kann mittels der IDE git-repos anlegen/clonen und auch aus der Oberfläche heraus
   commits starten und diese pushen.
   Weiter werden geänderte Zeilen markiert und bei einem Hover wird auch die aktuelle
   Änderung angezeigt.

3. Konsole
   Die IDE verfügt über eine Konsole, mit der man - ohne ein weiteres Fenster zu öffnen -
   bequem Pakete laden, git Befehle übergeben und vieles mehr machen kann.

Unterschiede zwischen JavaScript, PHP und C++
1.	Javascript wird von einem Interpreter eingelesen, C++ wird von einem Compiler compiliert.
2.	Javascript muss bei der Deklaration einer Variabel kein Datentyp angegeben werden, bei C++ schon.
3.	Javascript kennt fuer numerische Operationen nur den Datentyp Number, C++ kennt dagegen Interger,Float und Double.
4.	Javascript kann HTML manipulieren, C++ nicht.
5.	Javascript hat keinen festen Typ in einem Array, C++ schon.
6.	Javascript gibt Speicher frei mit einem Garbage-Collector, C++ muss Speicher freigeben manuell mit einem delete.
7.	Javascript hat keinen Destructor, C++ hat einen Destructor(bedingt durch 8)
8.	Javascript ist langsamer als C++ in der Ausfuehrung.
